## Why OpenNOW?

<CardGrid>
	<Card title="Native Performance" icon="rocket">
		Written in Rust with zero-overhead graphics bindings using wgpu and egui. No Electron, no web views.
	</Card>
	<Card title="Uncapped Potential" icon="open-book">
		No artificial limits on FPS, resolution, or bitrate. Stream at 240+ FPS if your connection supports it.
	</Card>
	<Card title="Privacy Focused" icon="approve-check">
		No telemetry by default. Your data stays on your machine.
	</Card>
	<Card title="Cross-Platform" icon="laptop">
		Native builds for Windows, macOS (Apple Silicon + Intel), and Linux.
	</Card>
</CardGrid>

## Quick Start

<Steps>
1. **Download** the latest release from [GitHub Releases](https://github.com/zortos293/OpenNOW/releases)

2. **Extract** and run the application
   - Windows: Extract zip, run `opennow-streamer.exe`
   - macOS: Extract zip, drag `OpenNOW.app` to Applications
   - Linux: Make AppImage executable and run it

3. **Login** with your NVIDIA GeForce NOW account

4. **Play** - Browse games and start streaming!
</Steps>

## Known Issues

<Aside type="danger" title="Performance Notice">
The app is being actively optimized. High CPU usage and stuttering are being addressed.
</Aside>

| Issue | Workaround |
|-------|------------|
| **High CPU usage** | Lower FPS/resolution. Optimization ongoing. |
| **Green screen flashes** | Switch to H.264 codec in settings. |
| **Audio stuttering** | Restart stream. Buffer improvements planned. |
| **Laggy input feeling** | Enable `low_latency_mode`. Being improved. |
| **Linux instability** | Use Windows/macOS for now. |
| **Raspberry Pi broken** | V4L2 decoder issues being investigated. |
